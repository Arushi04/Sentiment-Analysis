# Sentiment-Analysis

### List of Datasets
* Twitter Samples of NLTK

### Architecture without using ML libraries
* Logistic Regression
* Naive Bayes

### Requirements
* Python 3.6.10  
* Numpy 1.18.4  
* Pandas 1.0.5
* Matplotlib 3.2.1
* NLTK 3.5


### Results
Prediction accuracy using Logistic Regression : 99.5
Prediction accuracy using Naive Bayes : 99.4
